A personal task and calendar manager for Windows. Supports multiple user profiles, recurring tasks, and multiple color themes — all stored locally as JSON with no database needed.

Features

What it does

Multi-Profile
Separate profiles each with their own tasks and theme preference.
Calendar View
Monthly and weekly views with dot indicators on days that have tasks.
Recurring Tasks
Repeat tasks daily, weekly, or monthly with an optional end date.
Task Dashboard
See Today, Upcoming, and Backlog tasks in one organized view.
Themes
Multiple color themes saved per profile — switch anytime from settings.
Local Storage
All data saved as JSON on your machine. No internet or account needed.

Data Storage

Where your data lives

Everything is saved locally as JSON files — no account, no cloud, no database. The folder is created automatically on first launch.

# Location
C:\Users\<You>\AppData\Roaming\TodoCalendarApp\

# Files
profiles.json   → profiles (name, color, theme)
tasks.json     → tasks, recurrence, completion state
